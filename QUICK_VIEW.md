
## 💡 使用技巧

### API配置
1. 解析API可配置多个作为备用
2. 翻译API可配置不同模型
3. 配置会自动保存到浏览器
4. 下次访问自动加载

### 文件处理
1. 可先测试处理1-2页
2. 确认效果后处理全部
3. 大文件建议分批处理
4. 注意API额度限制

### 参数调整
1. OCR适用于扫描版PDF
2. 公式识别用于学术论文
3. 表格识别用于数据文档
4. 可按需启用/禁用

## 🔧 自定义

### 修改颜色
编辑 `static/css/style.css`:
```css
:root {
    --primary: #0ea5e9;  /* 改为你喜欢的颜色 */
}
```

### 修改文字
编辑 `templates/index.html`:
```html
<h1>翻译</h1>  <!-- 改为你想要的标题 -->
```
